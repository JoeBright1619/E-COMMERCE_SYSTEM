# 🏛️ Legacy Prototype

This directory contains the original, archived version of the CAGURA E-Commerce System.

## Why is this here?
Initially, this project was being built using **ASP.NET Core Razor Pages** for server-side HTML rendering. However, as the project evolved, the team made a strategic decision to transition to a modern **decoupled client-server architecture**.

We switched our backend to an **ASP.NET Core Web API** to serve JSON data, which allowed us to adopt **React** (via Vite) for a much more dynamic, responsive, and maintainable Single-Page Application (SPA) frontend.

## Purpose of this folder
This code is **no longer actively maintained** and should not be modified.

We are keeping it in the repository solely as a **visual and functional prototype reference**. Frontend developers should refer to the raw HTML, CSS, and layout structures within this folder to understand how the new React UI components should look and behave visually.
